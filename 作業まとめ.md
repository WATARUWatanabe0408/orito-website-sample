## v3（`v3/`）

### コンセプト
VEX Hero プロジェクトのデザイン言語を産業 B2B 文脈に適合させたフルダークテーマ。写真・動画不使用。CSS グラデーション + ドットグリッドで背景構成。

### カラーシステム
| 変数 | 値 | 用途 |
|---|---|---|
| `--bg-base` | `#080d0c` | ページ背景 |
| `--bg-surface` | `#0e1614` | セクション背景 |
| `--bg-card` | `#162220` | カード背景 |
| `--primary` | `#2dd4bf` | ティールアクセント |
| `--primary-dim` | `#0f6765` | サブアクセント |
| `--text-main` | `#f0fffe` | メインテキスト |
| `--text-sub` | `#8ba8a5` | サブテキスト |

### フォント
- 英字見出し: Inter（`letter-spacing: -0.04em`）
- 日本語: Noto Sans JP（800）

### 主要コンポーネント
- **Liquid Glass ナビ**: `backdrop-filter: blur(4px)` + グラデーションボーダー（全ページ共通）
- **Hero 背景**: ティールグロー放射グラデーション + ドットグリッド（32px）
- **文字アニメーション**: `data-char-animate` / `data-line` 属性、1文字ずつ translateX(-18px)→0、30ms 間隔
- **スクロール reveal**: IntersectionObserver、`.reveal` / `.is-visible`、`data-delay` で遅延制御
- **dark-card**: `#162220` 背景、ホバーでティールボーダー発光

### ファイル構成
```
v3/
  index.html      トップ（100vh Hero・3事業カード・選ばれる理由・会社案内・CTA）
  company.html    会社概要（60vh Hero・会社データテーブル・アクセス）
  service.html    事業紹介（65vh Hero・3サービスカード・CTA）
  contact.html    お問い合わせ（ダークフォーム・セレクト・送信ボタン）
  css/
    tailwind-config.js  ダークカラートークン定義
    common.css          Liquid Glass・hero-bg・reveal・dark-card
    top.css             golden spacing・strength-deco・company-brief-deco
    company.css         map-placeholder
    service.css         service-cta-bg
    contact.css         contact-input・contact-label・custom-checkbox
  js/
    main.js             文字アニメ・FadeIn・reveal・ハンバーガー
```

### テキストルール
`参考/` のテキストを一字一句変更しない。変更可能なのはマークアップ・クラス・スタイルのみ。
