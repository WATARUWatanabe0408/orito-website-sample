# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## プロジェクト概要

株式会社オリトのコーポレートウェブサイト。高圧ホース・油圧機器の卸・販売を行うB2B企業向けサイト。

**目的:** 会社ブランディング（A）＋問い合わせ獲得（B）

## サイト構成

```
index.html        トップページ
company.html      会社概要
service.html      事業内容
contact.html      お問い合わせ
css/
  style.css       共通スタイル
js/
  main.js         共通スクリプト
images/           イラスト・画像素材
legacy/           旧テンプレート（参考用、公開対象外）
```

## 技術スタック

- **実装:** バニラHTML/CSS/JavaScript（ビルドツール不要）
- **フォーム送信:** Formspree（サーバー不要でメール受信）
- **フォント:** Google Fonts（M PLUS Rounded 1c・Noto Sans JP）
- **アニメーション:** CSS + Intersection Observer（＋必要に応じてGSAP）
- **更新方式:** 静的HTML、更新は制作者が行う（CMS不要）

## デザイン方針

### カラーパレット

| 役割 | カラーコード | 用途 |
|---|---|---|
| メインカラー | `#3B7DD8` | ナビ・見出し・ボタン |
| アクセント | `#F97316` | CTAボタン・強調箇所 |
| 背景 | `#F5F7FA` | ページ背景 |
| テキスト | `#333333` | 本文 |

### フォント

- 見出し：**M PLUS Rounded 1c**（丸みがあり柔らかい印象）
- 本文：**Noto Sans JP**

### ビジュアル

- **写真は使わない**。イラスト中心で構成
- イラスト素材はフリー素材（tech-pic.com・loosedrawing.com・tyoudoii-illust.com等）のフラットイラスト系で統一
- 工場・機械・ホース・作業員などの題材で統一感を出す

### アニメーション

- ヒーローのイラスト：フェードイン
- 各セクション：スクロールでスライドアップ登場
- CTAボタン：ホバーで軽く浮き上がる

## 各ページの設計

### トップページ（index.html）

1. ヘッダー（ロゴ左・ナビ右）
2. ヒーロー（キャッチコピー＋サブテキスト＋問い合わせボタン＋イラスト）
3. 事業紹介3ポイント（アイコン＋短文 × 3カラム）
4. 事業内容ひとこと紹介（カード × 3）
5. お問い合わせCTA帯
6. フッター（住所・コピーライト）

### 会社概要（company.html）

- ページヘッダー（タイトル＋イラスト）
- 会社情報テーブル（会社名・所在地・代表者・設立・事業内容・電話・FAX等）
- Google Maps埋め込み

### 事業内容（service.html）

- ページヘッダー（タイトル＋イラスト）
- 事業カード × 6（2カラム）、各カード：イラスト＋タイトル＋説明
  1. 高圧ホース・口金・カプラ・スイベル・各種ホースの卸販売
  2. 高圧ホースアッセンブリーマシーンの販売
  3. 各種継手類の卸販売
  4. 高圧ホースの出張製作業務
  5. 洗浄用ホース・ノズル他の卸販売
  6. 油圧ボールバルブ・チェックバルブ・圧力計他油圧機器の販売

### お問い合わせ（contact.html）

- ページヘッダー
- フォーム（会社名・氏名・メールアドレス・お問い合わせ内容）
- Formspreeで送信処理
