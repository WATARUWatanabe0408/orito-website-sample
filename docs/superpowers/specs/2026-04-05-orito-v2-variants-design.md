# 株式会社オリト Ver.2（3バリアント）設計書

作成日: 2026-04-05

---

## 1. 概要

Ver.1（`/` 直下）のコーポレートサイトに加え、デザイン方向性の異なる3バリアントを `v2/` フォルダ以下に作成する。
まずトップページ（index.html）を各バリアントで作成して確認し、承認後に残りのページ（company / service / contact）を作成する。

### ディレクトリ構成

```
v2/
├── dark/          # 🌑 ダーク×インディゴ・重厚スタイル
│   ├── index.html
│   ├── company.html
│   ├── service.html
│   ├── contact.html
│   ├── css/style.css
│   └── js/main.js
├── pop/           # 🌈 ポップ×マルチカラー・gaaboo.jp寄り
│   ├── index.html
│   ├── company.html
│   ├── service.html
│   ├── contact.html
│   ├── css/style.css
│   └── js/main.js
└── scroll/        # 📖 スカイブルー×スクロールストーリー
    ├── index.html
    ├── company.html
    ├── service.html
    ├── contact.html 
    ├── css/style.css
    └── js/main.js
```

### 共通仕様

- バニラ HTML / CSS / JavaScript（ビルドツール不要）
- dark バリアントのみ GSAP（CDN）を使用
- pop・scroll は CSS アニメーション + Intersection Observer のみ
- フォーム送信: Formspree（各バリアント共通エンドポイント使用可）
- フォント: Google Fonts（バリアントごとに異なるフォントを使用）

## 2. 🌑 dark バリアント

### 2-1. カラーパレット

| 役割 | カラーコード | 用途 |
|---|---|---|
| 背景 | `#0f172a` | ページ全体の背景 |
| プライマリ | `#6366f1`（インディゴ） | 見出し・ボタン・アクセント |
| セカンダリ | `#10b981`（エメラルド） | 数字・バッジ・サブアクセント |
| テキスト | `#f1f5f9` | 本文（白寄り） |
| ミュート | `rgba(255,255,255,0.5)` | サブテキスト |
| カード背景 | `#1e293b` | カード・セクション背景 |

### 2-2. フォント

- 見出し: **Space Grotesk**（英字モダン）+ **Noto Sans JP**
- 本文: **Noto Sans JP**

### 2-3. トップページ セクション構成

```
① HERO（min-height: 100vh）
   - 左: キャッチコピー（大）+ サブテキスト + CTAボタン
   - 右: 大きなイラスト（GSAP で右から浮き上がりフェードイン）
   - 背景: 放射グラデーション（インディゴ + エメラルドのグロー）

② 数字で見るオリト（min-height: 60vh）
   - 背景: #1e293b
   - 取扱製品数・創業年・対応エリア数 を大きな数字で表示
   - スクロールイン時に 0 → 実数値へカウントアップアニメーション（GSAP）

③ 事業内容（カード3枚）
   - 暗い背景に光るボーダーカード
   - ホバーで インディゴのグロー エフェクト
   - スクロールイン時に下からフェードアップ

④ CTA（フルスクリーン）
   - 背景: GSAP パーティクル（小さい点が漂う）
   - 「お問い合わせはこちら」大ボタン
```

### 2-4. アニメーション仕様

| 要素 | アニメーション | ライブラリ |
|---|---|---|
| ヒーロー イラスト | 右 40px からフェードイン（1s, ease-out） | GSAP |
| ヒーロー テキスト | 下 30px からフェードアップ（0.8s, stagger） | GSAP |
| カウントアップ | 0 → 実数値（2s, ease-out） | GSAP |
| セクション全般 | スクロールイン時に下からフェードアップ | GSAP ScrollTrigger |
| カードホバー | インディゴ glow (box-shadow) | CSS |
| パーティクル | キャンバス上で点が漂う | Vanilla JS Canvas |

## 3. 🌈 pop バリアント

### 3-1. カラーパレット

| 役割 | カラーコード | 用途 |
|---|---|---|
| 背景 | `#fef9c3`（クリーム） | ページ全体の背景 |
| メイン | `#f59e0b`（アンバー） | ボタン・見出し・アクセント |
| タグ1 | `#10b981`（エメラルド） | 出張製作タグ |
| タグ2 | `#6366f1`（インディゴ） | 油圧機器タグ |
| タグ3 | `#ef4444`（レッド） | 強調タグ |
| テキスト | `#92400e` | 本文（ブラウン） |
| カード背景 | `#ffffff` | カード背景 |

### 3-2. フォント

- 見出し: **M PLUS Rounded 1c**（丸ゴシック・ポップ）
- 本文: **Noto Sans JP**

### 3-3. トップページ セクション構成

```
① HERO（min-height: 100vh）
   - 背景: クリームに大きな黄色い円がいくつか浮かぶ
   - 左: キャッチコピー（丸ゴシック・大）+ カラフルタグバッジ + CTAボタン
   - 右: 大きなイラスト + 絵文字が周囲に浮遊（ポヨン bounce アニメ）
   - イラスト・絵文字は CSS keyframes で個別に上下浮遊

② オリトの強み（3つ）
   - 白い丸カード × 3（水平並び）
   - 影は offset shadow（2px 2px 0 #d97706 スタイル）
   - スクロールイン時にポヨンと跳ねながら登場

③ 事業を絵で紹介（横スクロールカルーセル）
   - 6事業カードを横並びにして左右スクロール
   - マウスドラッグ／タッチスワイプ対応
   - カードホバーで上に跳ねる（translateY -8px）

④ CTA帯
   - 背景: アンバー（#f59e0b）
   - 「まずはお気軽に相談！」大ボタン（白）
   - 周囲に小さい絵文字がランダム配置
```

### 3-4. アニメーション仕様

| 要素 | アニメーション | ライブラリ |
|---|---|---|
| ヒーロー イラスト | スケール 0.8 → 1 + フェードイン | CSS |
| 絵文字の浮遊 | 上下 ±10px を 2〜4s でループ（個別 delay） | CSS keyframes |
| 強みカード登場 | translateY(40px) → 0 + opacity（stagger） | Intersection Observer + CSS |
| カードホバー | translateY(-8px) + shadow 強調 | CSS |
| カルーセル | マウスドラッグ / touch | Vanilla JS |

## 4. 📖 scroll バリアント

### 4-1. カラーパレット

| 役割 | カラーコード | 用途 |
|---|---|---|
| 背景 | `#f0f9ff`（スカイ） | ページ全体の背景 |
| プライマリ | `#0ea5e9`（スカイブルー） | 見出し・ライン・アクセント |
| テキスト | `#0c4a6e`（ディープネイビー） | 本文 |
| カード背景 | `#ffffff` | カード・セクション背景 |
| ライン | `#0ea5e9` | タイムライン縦線 |
| ミュート | `#94a3b8` | サブテキスト |

### 4-2. フォント

- 見出し: **Zen Kaku Gothic New**（細め・洗練）
- 本文: **Noto Sans JP**

### 4-3. トップページ セクション構成

```
① HERO（min-height: 100vh）
   - 左: キャッチコピー + サブテキスト + ボタン（テキストが1文字ずつフェードイン）
   - 右: イラスト（下からすっと上昇）
   - 左端に縦の青いライン（ページ全体のスクロール進行を示す装飾）

② 強みストーリー（交互レイアウト × 3）
   - 左テキスト右イラスト → 右テキスト左イラスト → 繰り返し
   - スクロールで左右それぞれの方向からスライドイン
   - セクション間をつなぐ曲線の SVG ライン

③ 事業タイムライン
   - 縦のタイムライン中央に線を引き、左右に事業カードを配置
   - スクロール進行に合わせてラインが上から下に伸びるアニメーション
   - 各カードはラインが到達したタイミングでフェードイン

④ CTA
   - シンプルな白背景 + 青いアクセント
   - 「お問い合わせ」ボタンのみ（装飾少なめ）
```

### 4-4. アニメーション仕様

| 要素 | アニメーション | ライブラリ |
|---|---|---|
| ヒーロー テキスト | 1文字ずつ opacity 0→1（stagger 0.05s） | CSS + JS |
| ヒーロー イラスト | 下 50px からフェードアップ | Intersection Observer + CSS |
| 交互レイアウト | 左右からスライドイン（translateX） | Intersection Observer + CSS |
| SVG ライン | stroke-dashoffset でパスが描かれる | CSS animation |
| タイムライン線 | scaleY 0→1（transform-origin: top） | Intersection Observer + CSS |
| タイムライン カード | ラインの伸長に遅延させてフェードイン | CSS transition-delay |

## 5. 内ページ共通仕様（各バリアント）

| ページ | 内容 |
|---|---|
| company.html | ページヘッダー + 会社情報テーブル + Google Maps プレースホルダー |
| service.html | ページヘッダー + 6事業カード |
| contact.html | ページヘッダー + Formspree フォーム |

各バリアントのデザインテーマ（配色・フォント・アニメーション）を内ページにも踏襲する。
内ページのページヘッダーはバリアントのプライマリカラーを使用。
