# 株式会社オリト コーポレートサイト 設計書

作成日: 2026-04-05

---

## 1. プロジェクト概要

### 会社情報
- **会社名:** 株式会社オリト
- **業種:** 高圧ホース・油圧機器の卸・販売（BtoB）
- **主要顧客:** 工場・製造業

### 事業内容
1. 高圧ホース・口金・カプラ・スイベル・各種ホースの卸販売
2. 高圧ホースアッセンブリーマシーンの販売
3. 各種継手類の卸販売
4. 高圧ホースの出張製作業務
5. 洗浄用ホース・ノズル他の卸販売
6. 油圧ボールバルブ・チェックバルブ・圧力計他油圧機器の販売

### サイトの目的
- **A. ブランディング:** 会社を知ってもらう・信頼感を伝える
- **B. リード獲得:** 問い合わせを増やす

## 4. デザイン仕様

### カラーパレット

| 役割 | カラーコード | 用途 |
|---|---|---|
| メインカラー | `#3B7DD8` | ナビ・見出し・ボタン |
| アクセント | `#F97316` | CTAボタン・強調箇所 |
| 背景 | `#F5F7FA` | ページ背景 |
| テキスト | `#333333` | 本文 |

### フォント

| 用途 | フォント |
|---|---|
| 見出し | M PLUS Rounded 1c（丸みがあり柔らかい印象） |
| 本文 | Noto Sans JP |

### ビジュアル方針

- **写真は使わない**
- イラスト中心（フリー素材のフラットイラスト系で統一）
- 素材候補: tech-pic.com（最優先）・loosedrawing.com・tyoudoii-illust.com等
- 題材は工場・機械・ホース・作業員など業種に合ったものに統一

### デザイントーン

- イラスト中心、柔らかく親しみやすい
- モダン・クリーン
- 動きがある（アニメーション）

### アニメーション仕様

| 要素 | アニメーション |
|---|---|
| ヒーローイラスト | フェードイン（ページロード時） |
| 各セクション | スクロールでスライドアップ登場 |
| CTAボタン | ホバーで軽く浮き上がる（translateY） |

## 5. 各ページ詳細設計

### 5-1. トップページ（index.html）

```
[ヘッダー]
  ロゴ（左）｜ナビゲーション（右）: 会社概要・事業内容・お問い合わせ

[ヒーロー]
  キャッチコピー（大）
  サブテキスト（会社・事業の一言紹介）
  「お問い合わせはこちら」ボタン（アクセントカラー）
  イラスト（右側に大きく配置）

[事業紹介3ポイント]
  アイコン＋短文 × 3カラム
  例: 豊富な在庫・迅速な対応・出張製作

[事業内容ひとこと紹介]
  カード × 3（主要カテゴリをピックアップ）
  「詳しく見る →」リンクで service.html へ

[お問い合わせCTA帯]
  「まずはお気軽にご相談ください」
  「お問い合わせ」ボタン

[フッター]
  会社名・住所・電話番号・コピーライト
```

### 5-2. 会社概要（company.html）

```
[ページヘッダー]
  ページタイトル「会社概要」＋イラスト

[会社情報テーブル]
  会社名 / 所在地 / 代表者 / 設立 /
  事業内容 / 電話番号 / FAX / 営業時間 等

[Google Maps]
  所在地の地図埋め込み
```

### 5-3. 事業内容（service.html）

```
[ページヘッダー]
  ページタイトル「事業内容」＋イラスト

[事業カード × 6（2カラム）]
  各カード構成: イラスト ＋ タイトル ＋ 説明文

  1. 高圧ホース・口金・カプラ・スイベル・各種ホースの卸販売
  2. 高圧ホースアッセンブリーマシーンの販売
  3. 各種継手類の卸販売
  4. 高圧ホースの出張製作業務
  5. 洗浄用ホース・ノズル他の卸販売
  6. 油圧ボールバルブ・チェックバルブ・圧力計他油圧機器の販売
```

### 5-4. お問い合わせ（contact.html）

```
[ページヘッダー]
  ページタイトル「お問い合わせ」

[フォーム]
  会社名（任意）
  氏名（必須）
  メールアドレス（必須）
  お問い合わせ内容（必須、テキストエリア）
  送信ボタン

[送信処理]
  Formspree を使用
  送信後: サンクスページまたはFormspreeのデフォルト完了画面
```

## 7. レスポンシブ対応

| ブレイクポイント | 対象 |
|---|---|
| 768px以上 | PC・タブレット |
| 767px以下 | スマートフォン |

- ヒーロー: PC=左テキスト右イラスト / SP=縦積み
- 事業カード: PC=2カラム / SP=1カラム
- 3ポイント: PC=3カラム / SP=1カラム
